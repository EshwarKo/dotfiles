# Dotfiles Project

Personal development environment configuration: nvim, tmux, and Claude Code.

## Structure

- `nvim/` - Neovim config (lazy.nvim, LSP, Treesitter, Telescope, gruvbox)
- `tmux/` - Tmux config (Ctrl-A prefix, vim-style navigation, gruvbox theme)
- `claude/` - Claude Code setup: roadmap, skills, scripts, Docker isolation

## Conventions

- Gruvbox dark theme across all tools
- Vim-style keybindings everywhere (hjkl navigation)
- Configs should be modular and well-commented
- Keep files concise; avoid bloat
